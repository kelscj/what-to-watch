# What to Watch

What to Watch is a Flask web application that helps users discover movies and TV shows based on keyword searches. The application returns relevant recommendations through a simple, user-friendly interface.

This project was built to practice full-stack web development using Python and Flask, with an emphasis on backend routing, user input handling, and dynamic content rendering.

---

# Features

* Search for movies and TV shows by keyword

* Displays relevant titles based on user input

* Backend implemented with Python and Flask

* Frontend built using HTML, CSS, and Jinja templates

# Getting Started

Follow the steps below to run the project locally.

# Prerequisites

* Python 3.x

* Git

# Installation and Setup

1. Clone the repository:  
    `git clone https://github.com/kelscj/what-to-watch.git`  
2. Navigate to the project directory:  
    `cd what-to-watch`  
3. Install dependencies:  
    `pip install -r requirements.txt`  
4. Run the data import script:  
    `python import_data.py`  
5. Start the Flask application:  
    `flask run`  
6. Open a browser and visit:  
    `http://127.0.0.1:5000`

---

# Notes

* The data import script only needs to be run once unless the database is reset.

* If the import script fails completely, verify that all dependencies are installed and that the database file exists in the project directory.

---

# Usage

Enter a keyword such as a genre or theme, submit the search, and view the recommended movies and TV shows returned by the application.
